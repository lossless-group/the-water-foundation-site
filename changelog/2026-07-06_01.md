# Dependency upgrade — Astro 6→7, @astrojs/vercel 10→11, @astrojs/svelte 8→9, Tailwind 4.3.1→4.3.2 + six malformed-HTML fixes

## Why Care?

Astro 7 ships Vite 8 and Rolldown as the default bundler — faster builds, stricter HTML validation. This site had accumulated six HTML nesting bugs that Vite's lenient compiler never reported. Rolldown caught every one of them. The good news: fixing them was mechanical; the bad news was finding them. Build is green.

## What's New?

| Package | Before | After |
|---|---|---|
| `astro` | `^6.4.0` | `^7.0.6` |
| `@astrojs/vercel` | `^10.0.8` | `^11.0.2` |
| `@astrojs/svelte` | `^8.1.0` | `^9.0.1` |
| `@tailwindcss/vite` | `^4.3.1` | `^4.3.2` |
| `tailwindcss` | `^4.3.1` | `^4.3.2` |

**Build result:** `Complete!` — 26+ pages rendered, server output, Vercel adapter, no errors.

## What had to be fixed

Six pre-existing malformed HTML bugs the Vite-based compiler never flagged:

1. **`component-library.astro:656`** — "Infographics/Data Display" section missing `<div class="component-category">` and `<div class="text-center mb-12">` wrappers. Inserted the two opening divs.

2. **`brand-kit/index.astro:103`** — `</BoilerPlateHTML>` closing tag referencing a component that no longer exists (renamed to `BaseThemeLayout`). Changed to `</BaseThemeLayout>`.

3. **`homepage2.astro:190`** — Orphaned `</div>` at the end of a water-animation section. Removed.

4. **`WhoWeAreExtendedV3.astro:120`** — `<div class="relative z-10 max-w-6xl">` opened at line 19 but never closed before `</section>`. Added missing `</div>`.

5. **`ColumnarMessages--Variant1.astro`** — Two issues: the outer `flex justify-center` div and the `relative z-10 max-w-7xl` wrapper both missing their closing tags. Added both.

6. **`TheBlindSpot.astro` (two copies)** — Both `meet-at/slides/` and `variant-1/slides/` variants had orphaned water-droplet animation fragments (leftover from a deleted visual panel). Removed the four orphaned lines in each.

7. **`HeroCenteredLogo--MobilePopOut.astro`** — Empty `<script></script>` tag. Rolldown attempts to build every script block; an empty block produces a "cannot find built path" error at prerender. Removed.

## Notes

- `@astrojs/svelte` 9 required no `legacy.componentApi` shim — the site's Svelte components are written in Svelte 5 runes syntax.
- Unsplash image loading errors during prerender (`403 Forbidden`) are pre-existing API key issues, not a regression from this upgrade.
